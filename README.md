# Card-pio-

Aplicação React com Vite para gerenciar e exibir cards de produtos/categorias. Inclui filtros por categoria, navegação e busca dinâmica.

## Demo

[Visite o site](https://mariachf.github.io/Card-pio-/)

## Funcionalidades

- Filtro por categorias (Principais, Sobremesas, Bebidas)
- Busca por nome de produto
- Cards responsivos com imagens e detalhes
- Navegação suave entre seções

## Tech Stack

| Tecnologia | Versão          |
|------------|-----------------|
| React      | ^19.1.1         |
| Vite       | rolldown-vite@7.1.14 |
| CSS Modules| -               |
| gh-pages   | ^6.3.0          |

## Instalação & Deploy

```bash
git clone https://github.com/Mariachf/Card-pio-.git
cd Card-pio-
npm install
npm run dev  # Desenvolvimento local
npm run deploy  # Deploy GitHub Pages

```
